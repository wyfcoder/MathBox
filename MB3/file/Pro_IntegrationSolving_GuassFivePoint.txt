Pro_IntegrationSolving_GuassFivePoint过程说明


原型：

 Pro_IntegrationSolving_GuassFivePoint(Function f,Number start,Number end);

 Pro_IntegrationSolving_GuassFivePoint(Function f,Number start,Number end，Number saved);


说明：

    该过程是一个用于求数值积分的过程算法。采用高斯五点算法算法原理。第一个参数Function是一个只有一个自变量的Function变量；第二个参数是一个Number类的表达式，表示积分下限；第三个参量也是一个Number类的表达式，表示积分上限；第四个变量是可以缺省的Number变量，用于保存积分结果。


#示例：

 Function f(x)=Cos(x+Sin(x))+Sin(x+Cos(x));

 Number a=20;

 Pro_IntegrationSolving_GuassFivePoint(f,-10,10+a,a);


#输出：

 >>New Function 

 @ f=Cos(x+Sin(x))+Sin(x+Cos(x))

 >>New Number 

 @ a=20

 >>Solve the integration, the answer is -16.4849753729468098

 >>Rewrite Number 
 
 @ a=-16.485
